# Plex-Donate
This app lets you:
- Collect **PayPal subscriptions** (monthly donation).
- Auto-create a **Wizarr invite** and email the donor.
- Manage everything from an **Admin dashboard**.
- Optionally revoke Plex access if a subscription is canceled.
